# 📊 XP System - Komplette Übersicht

## 🎯 Wie bekomme ich XP?

### 1️⃣ Nachrichten schreiben (Hauptquelle)
- **15 XP pro Nachricht**
- **60 Sekunden Cooldown** zwischen XP-Gains
- Funktioniert in ALLEN Channels
- Bots bekommen keine XP

**Beispiel-Timeline:**
```
16:00:00 → Nachricht schreiben → +15 XP ✅
16:00:30 → Nachricht schreiben → Kein XP (Cooldown) ❌
16:01:00 → Nachricht schreiben → +15 XP ✅
16:02:00 → Nachricht schreiben → +15 XP ✅
```

**Tipps:**
- Aktiv im Chat sein!
- Nicht spammen (wegen Cooldown sinnlos)
- Normal schreiben = automatisch XP

---

### 2️⃣ Coinflip Gewinn (Bonus-XP!)
- **5 XP pro gewonnenem Coin!**
- Nur bei Gewinn
- KEIN Cooldown!

**Beispiel:**
```
/betlabcoinflip 50
→ 🎉 GEWONNEN!
→ +50 Coins
→ +250 XP Bonus! 🎁

(50 Coins × 5 = 250 XP!)
```

**Strategie:**
- Kleine Einsätze für sicheren XP-Flow
- Große Einsätze für XP-Boost Risiko

## 💡 XP-Optimierung Tipps

### Für schnelles Leveln:
1. **Aktiv chatten** (alle 60s eine Nachricht = max XP)
2. **Coinflip nutzen** (Gewinn = Bonus-XP!)
3. **Nicht spammen** (Cooldown macht es sinnlos)

### Für Coin-Farming:
1. **Niedrige Level** = schnelle Level-Ups = viele Coins
2. **Auf Milestones warten** (Level 25/50/75/100)
3. **Coinflip-Gewinne** reinvestieren
